Tsuki is a minimalistic open-sourced social media platform, built using Python.


About

Tsuki (つき) is a noun meaning the moon in Japanese. The Tsuki project aims to be simple yet graceful like the moon. It is an easy to use minimal social media platform.

We chose this name due to the core developer's favourite quote, つきがきれいですね meaning the moon is beautiful, isn't it?


Contribute

The Tsuki project is open for contributions on GitHub.
